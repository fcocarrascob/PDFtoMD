Documento de Prueba: Fase 1 y Fase 2

Este documento prueba todas las funcionalidades implementadas en Fase 1 (funciones) y Fase 2 (arrays).

Fase 1: Funciones Definidas por Usuario

1.1 Función simple de un parámetro

$$ f(x) = 1 + x^{2} + 2 \cdot x $$
= Function f(x) defined
$$ resultado1 = f{\left(3 \right)} $$
= 16.00

1.2 Función con múltiples parámetros

$$ area_rectangulo(base, altura) = altura \cdot base $$
= Function area_rectangulo(base, altura) defined
$$ A = \operatorname{area}_{rectangulo}{\left(5,3 \right)} $$
= 15.00

1.3 Función con operaciones trigonométricas

$$ g(t) = \cos{\left(t \right)} + \sin{\left(t \right)} $$
= Function g(t) defined
$$ resultado2 = g{\left(\frac{\pi}{4} \right)} $$
= 1.41

1.4 Función que usa variables globales

$$ constante = 10.0 $$
= 10.00
$$ h(x) = constante + x $$
= Function h(x) defined
$$ resultado3 = h{\left(5 \right)} $$
= h(5)

1.5 Composición de funciones

$$ cuadrado(x) = x^{2} $$
= Function cuadrado(x) defined
$$ doble(x) = 2 \cdot x $$
= Function doble(x) defined
$$ compuesta(x) = \operatorname{cuadrado}{\left(\operatorname{doble}{\left(x \right)} \right)} $$
= Function compuesta(x) defined
$$ resultado4 = \operatorname{compuesta}{\left(3 \right)} $$
= compuesta(3)

Fase 2: Arrays con linspace y arange

2.1 linspace básico - Array pequeño (visualización completa)

$$ arr1 = \operatorname{linspace}{\left(0,10,5 \right)} $$
= Array: [0.00, 2.50, 5.00, 7.50, 10.00]

2.2 arange básico - Array pequeño

$$ arr2 = \operatorname{arange}{\left(0,10,2 \right)} $$
= Array: [0.00, 2.00, 4.00, 6.00, 8.00]

2.3 linspace con valores negativos

$$ arr3 = \operatorname{linspace}{\left(-5,5,5 \right)} $$
= Array: [-5.00, -2.50, 0.00, 2.50, 5.00]

2.4 arange con paso decimal

$$ arr4 = \operatorname{arange}{\left(0,5,0.5 \right)} $$
= Array (10 values): [0.00, 0.50, 1.00, ...]

2.5 linspace grande (visualización truncada)

$$ arr5 = \operatorname{linspace}{\left(0,100,50 \right)} $$
= Array (50 values): [0.00, 2.04, 4.08, ...]

2.6 Arrays usando variables

$$ inicio = 0.0 $$
= 0.00
$$ fin = 20.0 $$
= 20.00
$$ puntos = 11.0 $$
= 11.00
$$ arr6 = \operatorname{linspace}{\left(inicio,fin,puntos \right)} $$
= Array (11 values): [0.00, 2.00, 4.00, ...]

2.7 Arrays con expresiones matemáticas

$$ arr7 = \operatorname{linspace}{\left(6,25,10 \right)} $$
= Array (10 values): [6.00, 8.11, 10.22, ...]
$$ arr8 = \operatorname{arange}{\left(0,2 \cdot \pi,\frac{\pi}{4} \right)} $$
= Array (8 values): [0.00, 0.79, 1.57, ...]

2.8 arange con paso negativo (descendente)

$$ arr9 = \operatorname{arange}{\left(10,0,-1 \right)} $$
= Array (10 values): [10.00, 9.00, 8.00, ...]
